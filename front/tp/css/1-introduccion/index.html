Introducción a CSS

¿Que es css?

Es el lenguaje que utilizamos para definir la apariencia visual de un sitio web.

¿Para que sirve css?

  • Estilizar
    • Cambiar colores, tamaños, tipos de letra y espaciados para mejorar la legibilidad y la identidad visual.
  • Posicionar
    • Organizar los elementos en la pantalla de forma precisa, creando estructuras visuales coherentes.
  • Animar
    • Crear transiciones, efectos visuales y animaciones que mejoran la experiencia del usuario.
  • Adaptar
    • Diseñar interfaces responsivas que se ajusten automáticamente a diferentes tamaños de pantalla.


CSS (Cascading Style Sheets) es un lenguaje de estilo que se utiliza para describir la presentación de un documento HTML. Permite separar el contenido de la presentación, lo que facilita la creación de sitios web visualmente atractivos y fáciles de mantener.

Con CSS, los desarrolladores pueden aplicar estilos a elementos HTML de manera eficiente, utilizando selectores y propiedades para definir cómo se deben mostrar los elementos en la página. Esto incluye la posibilidad de cambiar colores, fuentes, márgenes, y mucho más, todo sin modificar el contenido HTML subyacente.

En resumen, CSS es una herramienta poderosa que permite a los desarrolladores web crear experiencias visuales atractivas y funcionales, mejorando la usabilidad y la estética de los sitios web.
